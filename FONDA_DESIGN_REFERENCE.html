07 — Illustration System (SVG Pencil + Watercolor, hotel objects)

All illustrations are SVG. Pencil stroke + watercolor wash. Stroke color: #3D3528. Fills: warm low-opacity washes. Placed at section corners, never center-stage. 160–240px wide. Always opacity: 0.8–0.9.

Morning coffee
Hero / CTA sections

AI Chat / Features
Concierge bell
Check-in section
Envelope
Email feature
Briefing notepad
Problem / Morning section

Illustration rules for Claude Code: All SVGs use stroke color #3D3528, stroke-width 1.4–1.8, stroke-linecap/linejoin: round. Fills are low-opacity (opacity: 0.3–0.65) warm washes: #F5EDE8, #E8C99A, #FAFAF8. Always add a ground shadow ellipse at the bottom (fill="#3D3528" opacity="0.05"). Never use flat fills or pure geometric shapes — paths should feel hand-drawn. Place at section corners with position: absolute, never in the content flow.

12 — Quick Reference Card (paste at top of any Claude Code prompt)

# Fonda Design System — Quick Reference

FONTS: Playfair Display (headlines) · Inter (body/UI)

BG: #FAFAF8 (page) · #F4F2EE (cards) · #FFFFFF (modals only)

TEXT: #1C1915 (primary) · #6B6057 (secondary) · #A09488 (muted)

ACCENT: #1E3A5F navy · hover #162D4A · light bg #EBF1F8

INK: #1C1915 (dark pill buttons, like Sana.ai)

BORDER: #E2DDD5

RADIUS: 100px (buttons) · 20px (cards) · 12px (inputs) · 28px (CTA box)

HEADLINES: Playfair weight 400, italic second line always

BUTTONS: Always pill · navy OR ink · ghost for secondary

ILLOS: SVG pencil+watercolor · hotel objects · 160–240px · corner-placed

MOTION: fade-up scroll · translateY(-2px) hover · no parallax

LIGHT ONLY — no dark mode. Never pure #000 or #FFF on page.
